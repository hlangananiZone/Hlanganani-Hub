# Hlanganani-Hub

## Overview
A free, accessible platform helping South Africans apply for NSFAS, SASSA, universities, colleges, clinics, and stay informed through trusted resources.

## Purpose
This project exists to simplify access to essential government and education services for young people by creating a single, easy-to-use digital hub.

## Features
- NSFAS application guidance  
- SASSA application support  
- University & college applications (South Africa)  
- Clinic and health resource links  
- Youth-focused information hub  
- Simple and accessible interface  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Setup Instructions
1. Clone the repository  
2. Open `index.html` in any browser  
3. No installation required for basic version
